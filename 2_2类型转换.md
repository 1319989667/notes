# 类型转换

# C 语言风格

1. 隐式类型转换

```
double value = 5.25;
int a = value;
```

2. 显式类型转换

```
double value = 5.25;
int a = (int)value;
```



## C++风格

注意，C++风格能做到的事情用C语言风格也能做到,它其实是一种语法糖，会在编译时进行检查。

1. static_cast

   将进行编译检查。

```
double value = 5.25
double s = static_cast<int>(value) + 5.3;
```

2. reinterpret_cast

​	重新解释强制转换。

3. const_cast

   添加或删除const。

4. dynamic_cast

   将进行运行时检查。经典例子是尝试将基类指针转换为派生类。

   ```
   Base* base = new Derived();
   if(dynamic_cast<Derived*>(base))
   {
   	std::cout << "success!\n";
   }
   ```
